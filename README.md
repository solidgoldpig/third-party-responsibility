# fb-ioj

IOJ service

## Pre-requisites

  [Node](https://nodejs.org)

## Installing

```
git clone git@github.com:ministryofjustice/fb-ioj.git

git clone git@github.com:ministryofjustice/fb-runner-node.git
cd fb-runner-node
npm install
```

## Usage

```
cd fb-runner-node

SERVICEDATA=<PATHTO>/fb-example-service npm start
```
